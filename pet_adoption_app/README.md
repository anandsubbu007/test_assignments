# Pet Adoption App - Assignment

A Pet Adoption appllication using Flutter.


## Getting Started

### To Debug App
 - flutter pub get
 - flutter run

### To Run Test
 - Unit Test
    - flutter test test/bloc_test.dart
    - flutter test test/storage_test.dart
 - Widget Test
     - flutter test test/widget_test.dart
